# Iced Moving Picture

`iced-moving-picture` is an Iced widget library
(forked from [iced_gif](https://github.com/tarkah/iced_gif))
to show animated images using [Iced](https://github.com/iced-rs/iced).

Supported formats:

* GIF
* APNG
